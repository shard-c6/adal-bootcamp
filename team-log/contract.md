# Contract — HydraDB Hero Clone

Status: Approved to Build

Clone the hero section of the landing page, https://www.hydradb.com, put it inside of hyperspell-clone, needs to be exactly the same, from animation, content, font, color, styling, everything, pixel level, and video level including all effects.

## Scope
- Build only the top landing-page hero experience in `hyperspell-clone`.
- Include matching desktop/mobile layout, typography, colors, content, hero imagery, nav, background styling, and entrance/ambient animation effects.
- Use the supplied docs: `clone_guide_v2.md` as operational guide and `clone_landing_page_101.md` as deep reference.

## Acceptance Criteria
1. App lives in `hyperspell-clone`.
2. Hero copy matches HydraDB visible hero: "The Graph AI Runs On.", storage/value subtitle, and build-on ontology/memory/company-brain/context-graph line.
3. Nav/header visually matches HydraDB dark Framer landing style.
4. Uses dark near-black background with subtle green/blue radial glow/noise impression.
5. Uses HydraDB remote image assets from observed `framerusercontent.com` hero image URLs.
6. Hero media/cards are arranged in a floating collage matching the target impression.
7. Entrance animation exists: nav, headline, copy, CTA/media fade/translate in staggered timing.
8. Ambient/video-level effects exist: floating cards, glow pulse, shimmer/grid, and hover motion.
9. Typography uses a close modern sans stack and tight letter spacing matching target.
10. Responsive behavior covers desktop wide, 1440 desktop, and mobile hero layout.
11. Build must pass.
12. No unrelated repository files are modified except coordination docs and the new app.
